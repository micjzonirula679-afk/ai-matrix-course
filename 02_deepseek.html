第01节 DeepSeek注册+账号保护完整流程
✅ 学完后你能获得：成功注册账号，避开封号坑，账号稳定可用

注册步骤

打开浏览器，访问 deepseek.com，点击右上角"注册"按钮。注册方式有两种：手机号注册和邮箱注册。国内用户推荐直接用手机号，输入手机号、获取验证码、设置密码，30秒完成注册。

注册完成后，会自动进入对话界面，这就是你以后用来写文案、写脚本、生成标题的主要工作台。

账号保护注意事项

  1. 不要在任何第三方网站用DeepSeek账号登录。网上有很多"DeepSeek套壳网站"，打着免费用的旗号，实际上是收集账号信息的。认准官方域名 deepseek.com，其他域名一律不用。
  2. 不要把账号分享给别人。DeepSeek目前对账号有使用频率限制，多人共用一个账号容易触发风控，导致账号被封或者功能受限。
  3. 密码设置要有一定复杂度，不要用生日、手机号这类容易被猜到的密码。
  4. 高峰期DeepSeek服务器可能会拥挤，出现"服务繁忙"的提示。这不是你的账号有问题，等一会儿或者换个时间段再试就好。
第02节 DeepSeek正确使用方法（界面精讲）
✅ 学完后你能获得：玩转所有功能按钮，比身边90%的人用得更溜

界面核心区域

左侧：历史对话记录，你每次开启的对话都会保存在这里，方便随时回来继续。建议给每次对话起个名字，比如"小红书文案"、"抖音脚本"，方便管理。

中间：对话区域，这是你主要工作的地方。

底部：输入框，在这里输入你的需求，按回车或者点击发送按钮。

两个重要功能

第一个是"深度思考"模式。在输入框左下角有一个开关，开启之后DeepSeek会花更多时间思考，输出质量更高，适合用来写比较复杂的脚本或者策划方案。普通文案不需要开，节省时间。

第二个是"联网搜索"功能。开启之后DeepSeek可以搜索最新的网络信息，适合用来找热点、查数据、了解最新趋势。写追热点的文案时记得开启这个功能。

正确的使用习惯

不要只发一句话就等结果。DeepSeek给出的第一版内容往往是通用版，你需要继续告诉它"改得更口语化一点"、"加一个反转结尾"、"字数控制在150字以内"，经过2到3轮对话，才能得到真正好用的内容。

把DeepSeek当成一个实习生，你是主管。你给的指令越清晰，它交出来的作业就越好。
第03节 DeepSeek写提示词核心公式（附模板）
✅ 学完后你能获得：掌握万能提示词公式，每次提问都能出好内容

提示词就是你给DeepSeek的指令。指令写得好，出来的内容就好；指令写得烂，出来的都是废话。

万能提示词公式

你是谁 + 帮我做什么 + 具体要求 + 输出格式

举个例子：

❌ 差的提示词：帮我写一条视频文案

✅ 好的提示词：你是一位擅长写抖音爆款文案的短视频运营专家。帮我写一条关于"上班族副业赚钱"的短视频开场文案，要求：前3秒要有强烈的钩子，让人不想划走；语言口语化，不要书面语；字数控制在80字以内。

两种写法，出来的结果天差地别。

5个实用提示词模板

模板一（爆款文案）：
你是抖音爆款文案写手。帮我写一条关于【主题】的短视频文案，风格【搞笑/治愈/励志】，目标用户是【人群】，要求开头3秒抓人，结尾引导互动，字数【X】字以内。
模板二（短视频脚本）：
你是短视频导演。帮我写一条【时长】秒的短视频脚本，主题是【主题】，格式包含：画面描述、旁白文字、字幕内容，风格参考【某类账号风格】。
模板三（爆款标题）：
帮我给以下视频内容生成10个抖音爆款标题，要求有悬念感或者数字感，让人想点进来看：【粘贴你的视频内容简介】
模板四（评论回复）：
帮我写10条回复粉丝评论的话术，风格亲切有趣，适合在抖音账号下互动用，主题是【账号方向】。
模板五（选题策划）：
我做的是【账号方向】，帮我策划20个近期可以做的短视频选题，要求贴近热点、有传播性、适合普通人拍摄。
把这5个模板保存下来，日常工作直接套用，填空就行。
第04节 DeepSeek写爆款文案实战（10种类型）
✅ 学完后你能获得：能写出钩子感强、让人想看完的短视频文案

短视频文案不是随便写的，不同类型的文案有不同的结构逻辑。掌握这10种类型，基本覆盖抖音小红书90%的爆款文案形式。

类型一：痛点型

先说出目标用户最难受的那件事，让他觉得"说的就是我"。

示例提示词：帮我写一条针对"每天上班很累但工资很低"这个痛点的短视频开场文案，要让人看了心里一紧。

类型二：悬念型

开头留一个问题，让人非得看完才知道答案。

示例提示词：帮我写一条悬念型文案，主题是"我用AI一个月赚了多少钱"，开头要让人好奇，结尾揭晓答案。

类型三：数字型

用具体数字增加可信度和冲击感。示例：3天涨粉1万、月入2万的副业、0成本启动的生意……数字越具体越有说服力。

类型四：反常识型

说一个跟大多数人认知相反的观点，制造认知冲突。

示例提示词：帮我写一条反常识型文案，主题是"努力工作不一定能赚到钱"，要有观点、有态度。

类型五：干货型

直接给有用的信息，让人觉得"这条视频我没白看"。

示例提示词：帮我写一条干货型文案，主题是"用DeepSeek写文案的3个技巧"，条理清晰，语言简洁。

类型六：共鸣型

说出某类人群的真实心声，让人忍不住转发给朋友看。

示例提示词：帮我写一条让上班族看了想转发的共鸣型文案，主题是"明明很努力却感觉原地踏步"。

类型七：对比型

用前后对比、两种选择的对比，制造强烈反差。示例：用AI之前 vs 用AI之后、普通人的一天 vs 有副业的人的一天。

类型八：故事型

用第一人称讲一个真实感强的小故事，带入感强，完播率高。

示例提示词：帮我用第一人称写一个故事型短视频脚本，主题是"我辞掉工作开始做AI副业的经历"，200字左右，真实感要强。

类型九：福利型

承诺给用户某种好处，驱动关注和互动。示例：评论"想要"我发你、点赞收藏我私信你资料包……

类型十：热点蹭流量型

结合当下热门话题，借势流量。

示例提示词：现在最热的话题是【XX】，帮我结合这个热点写一条关于AI副业的短视频文案。
第05节 DeepSeek写短视频脚本（口播/图文/剧情）
✅ 学完后你能获得：10分钟出一条完整短视频脚本，效率提升10倍

脚本是视频的骨架。脚本写好了，视频就成功了一半。

口播脚本

适合真人出镜讲话的视频。结构：开场钩子（3秒）→ 核心内容（30-50秒）→ 结尾行动指令（5秒）。

口播脚本提示词模板：
帮我写一条60秒口播脚本，主题是【主题】，目标用户是【人群】，风格【接地气/专业/幽默】，开头必须在3秒内抓住注意力，结尾引导关注，格式输出为：时间轴 + 口播文字 + 镜头提示。

图文脚本

适合AI生图+配音的视频，不需要真人出镜。结构：第1张图（开场）→ 第2-5张图（内容展开）→ 最后一张图（结尾引导）。

图文脚本提示词模板：
帮我写一条图文短视频脚本，主题是【主题】，共6张图，每张图包含：图片画面描述 + 配音文字，整体时长约40秒，风格【治愈/励志/搞笑】。

剧情脚本

适合有故事情节的视频，完播率最高。结构：冲突开场 → 矛盾发展 → 反转或解决 → 结尾升华。

剧情脚本提示词模板：
帮我写一个短剧脚本，主题是【主题】，时长约90秒，包含2个角色，剧情要有反转，格式输出为：场景描述 + 角色对话 + 镜头说明。
实用技巧：写完脚本之后，让DeepSeek再帮你做一件事——"帮我检查这个脚本，找出可能让观众划走的地方，并给出修改建议。"这一步能让脚本质量再提升一个档次。
第06节 DeepSeek一键生成爆款标题（含公式拆解）
✅ 学完后你能获得：生成高点击率标题，让视频在推荐页抢眼球

视频做得再好，标题没写好，播放量也上不去。标题是决定算法给不给你推流的关键因素之一。

爆款标题的4个核心要素

  1. 有关键词。平台算法靠关键词识别你的内容，推给对应的用户。做AI副业内容，标题里要出现"AI"、"副业"、"赚钱"这类词。
  2. 有情绪。让人看了标题有感觉，想点进去。"月入过万"比"赚了一些钱"更有冲击力。
  3. 有悬念或者好奇心缺口。"我用这个方法一个月赚了X元（附教程）"比"AI赚钱教程"更让人想点。
  4. 字数控制。抖音标题建议20字以内，小红书可以稍长，但也不超过30字。

6种爆款标题公式

公式一（数字+结果）：X天/X元/X个方法 + 具体结果
示例：学了3天DeepSeek，上周接了4单赚了800
公式二（反问+痛点）：你还在……吗？
示例：你还在用脑子想文案？AI10秒帮你写好
公式三（悬念+揭秘）：为什么……（答案在视频里）
示例：为什么同样做AI视频，有人月入过万有人0收入
公式四（对比+冲击）：普通人 vs 会用AI的人
示例：不会AI的人做视频要3小时，我只需要10分钟
公式五（身份+共鸣）：专门写给……的人
示例：写给还在上班想做副业的普通人
公式六（福利+行动）：免费分享……（限时/限量）
示例：免费分享我用了半年的DeepSeek提示词模板
生成标题的提示词：
帮我给以下视频内容生成15个适合抖音/小红书的爆款标题，要求：有点击欲望、包含核心关键词、字数20字以内，视频内容是：【粘贴内容简介】
一次生成15个，从里面挑最好的1到2个用。
第07节 DeepSeek做情绪文案（治愈/励志/毒鸡汤）
✅ 学完后你能获得：写出让人收藏转发的情绪共鸣型爆款文案

情绪类文案是小红书和抖音上最容易爆的内容类型之一。因为它不需要干货，不需要技巧教学，只需要说中用户的心里话，就能触发大量收藏和转发。

治愈型

让疲惫的人感到被理解和安慰。适合晚上10点到12点发布，这个时间段用户情绪最敏感。

治愈型提示词：
帮我写一段治愈型短视频文案，主题是"累了就休息一下，不用对自己那么苛刻"，语气温柔，像朋友说话，100字左右，结尾要让人想保存。

励志型

给人打气，让人看完想行动。适合早上7点到9点发布，配合用户上班路上的心理状态。

励志型提示词：
帮我写一段励志型短视频文案，主题是"普通人也能靠努力改变命运"，有力量感但不说教，100字左右，开头要有冲击力。

毒鸡汤型

表面看起来是鸡汤，实际上是反话或者扎心的真相。完播率极高，因为让人又好气又好笑。

毒鸡汤提示词：
帮我写5条毒鸡汤文案，主题围绕"职场/赚钱/奋斗"，语气要犀利有点毒，让人看完觉得"哈哈哈说得对"，每条30字以内。

情绪文案的变现逻辑

情绪文案账号涨粉快、互动高，但直接变现能力相对弱。最佳组合是：情绪文案涨粉 → 偶尔穿插AI工具/副业教程内容 → 引流到私域 → 卖课或者接单变现。这是很多小红书博主正在用的路径，亲测有效。
